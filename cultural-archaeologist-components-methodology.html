Cultural Archaeologist: Component Methodology

Public one-page reference for the current methodology used across all eight Cultural Archaeologist components.

Audience baseline used in QA pages: Gen Z

Moments

  • Deliver 6-10 insights balancing breaking shifts (last 7 days) with structural macro forces.
  • Prioritize hard macro signals over soft speculation.
  • Only layer psychology when directly tied to macro context.

Beliefs

  • Deliver 6-10 stabilized values/perceptions from highly corroborated multi-month evidence.
  • Ignore fleeting daily controversies and reaction spikes.
  • Optionally include a recent topic only when explicitly labeled as recent.

Tone

  • Deliver 6-10 sustained emotional-baseline insights using lexical sentiment patterns.
  • Use the same stabilized digest standard as Beliefs.
  • Map each insight to an archetype spectrum (for example: Stable vs Reactive).

Language

  • Capture how the audience communicates through vernacular, symbols, and code.
  • Prioritize current usage with practical copywriter relevance.
  • Flag known/inferred/speculative language confidence where needed.

Behaviors

  • Deliver 6-10 behavior insights with an up-to-date (not most-recent-only) lens.
  • Balance consistent rituals/habits with recent fads/challenges.
  • Weight repeatable actions over one-off virality.

Contradictions

  • Deliver 6-10 tensions by cross-referencing Beliefs (what they say) and Behaviors (what they do).
  • Use exact narrative structure: What they say / What they do / Tension.
  • Include evidence-type framing and avoid uncoupled speculation.

Community

  • Deliver 6-10 identity anchors using a barbell mix.
  • Include foundational long-standing hubs and breakout micro-communities from ~last 30 days.
  • If exact names are uncertain, fallback to platform location (Reddit, Discord, Substack).

Influencers

  • Deliver 6-10 influencers with a barbell mix: 3-4 legacy authorities + 3-4 breakout micro-creators.
  • Evaluate via Resonance (growth speed), Conversion (niche fit), and Penetration (visibility).
  • If Social Blade is integrated, query both Top Followers and Highest 30-Day Growth.
  • If naming certainty is weak, explicitly say so and do not invent names.
